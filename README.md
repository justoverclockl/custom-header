A [Flarum](http://flarum.org) extension. Add a Custom Header to Flarum

### Installation

Install with composer:

```sh
composer require justoverclock/custom-header:"*"
```

### Updating

```sh
composer update justoverclock/custom-header:"*"
php flarum migrate
php flarum cache:clear
```
